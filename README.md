# First Thing First
> Clone / Download first :D

> Change .env.example to .env then update env

> composer update

> php artisan migrate:fresh --seed

> composer require laravel/ui --dev

> php artisan ui vue --auth

> npm install / update

> npm run watch

> composer dump-autoload

> php artisan serve
